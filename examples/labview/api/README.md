### Important Note
For 64Bit version of LabView please use the library in "x64" and build new VIs based on 32Bit samples. The exampes was created with LabView 2011. The iowkit.dll can be found [HERE](https://github.com/codemercs-com/io-warrior-win/tree/main/dll).

### Known issues
Based on the version of LabView (x86 or x64) you have to use the right library and iowkit.dll.
Otherwise LabView or the VI will crash. If this happens you have to disconneted the IO-Warrior and stop the LabView process by using the task manager of Windows.
